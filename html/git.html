Git

  • git init – Инициализация локального репозитория.
  • git add – Добавление файлов в индекс (подготовка к фиксации изменений).
  • git status – Проверка статуса репозитория и отслеживаемых файлов.
  • git commit – Фиксация изменений (создание коммита в истории проекта).
  • git push – Отправка зафиксированных изменений в удаленный репозиторий.
  • git pull – Получение и слияние изменений из удаленного репозитория.
