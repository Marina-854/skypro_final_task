Git

  • git init - инициализировать локальный Git-репозиторий в текущей папке.
  • git clone [url] - клонировать (скачать) удаленный репозиторий на свой компьютер.
  • git status - проверить текущее состояние репозитория и список измененных файлов.
  • git add . - добавить все измененные файлы в индекс для подготовки к коммиту.
  • git commit -m "сообщение" - сохранить зафиксированные изменения с понятным описанием.
  • git push origin main - отправить сохраненные локальные коммиты в удаленный репозиторий на GitHub.
